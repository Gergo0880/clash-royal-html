## Leírás
Ez a repo tartalmaz egy **egyoldalas, Bootstrap 5 alapú** HTML/CSS/JS weboldalt, amely:
- részletesen leírja a Clash Royale játékmenetét;
- felsorolja a fő stratégiákat és tippeket;
- beépített, sötét neon vizuális stílust használ;
- tartalmaz SVG ikonokat (harc, pajzs, elixir, korona, stb.);
- az oldal alján egy 5–6 mondatos szerelmes sztori van (király + hercegnő dráma).

A cél: demo/prototípus, portfólió, vagy statikus infóoldal.

---

## Funkciók
- Teljesen reszponzív Bootstrap 5 layout
- Sötét, neon-stílusú design
- Inline SVG ikonok és egyenletes neon glow effektek
- Egyszerű JS interakciók: aktív nav, kártya-szűrő példa
- Placeholder YouTube beágyazás és letöltés gombok (Google Play / App Store SVG-ek)
- Könnyen testreszabható szöveg és assetek
